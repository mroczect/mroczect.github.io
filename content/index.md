# Muhammad Riduwan Khafidi

## Profile

I am **Muhammad Riduwan Khafidi**, an **18**-year-old student in D4 **Software Engineering Techniques** (`TRPL`) at Batam, class of 2026. I have about 4 years of programming experience, focused primarily on `Zig`, `Rust`, and `C`.

My work centers on libraries, frameworks, and developer tools. I care about building software that is:

- Small
- Reusable
- Easy to integrate without unnecessary complexity

> [!NOTE]
> **Principles**
>
> - Minimal and explicit APIs
> - Predictable behavior
> - Strong validation
> - Low magic, high clarity

Good tools should be easy to understand, hard to misuse, and simple to integrate.

## Featured Projects

### 1. RAWSSG

A terminal‑themed static site generator for blogs and project documentation, written in Rust.  
It compiles Markdown files into a complete static site with a built‑in development server, live reload, and a man‑page aesthetic.

- Repository: [https://github.com/mroczect/rawssg](https://github.com/mroczect/rawssg)
- Demo: [https://mroczect.github.io/rawssg/](https://mroczect.github.io/rawssg/)

### 2. Velodiff

A library for generating diffs of a Git repository, written with efficiency and minimalism in mind.

- Repository: [https://github.com/mroczect/velodiff](https://github.com/mroczect/velodiff)
- Docs: [https://mroczect.github.io/velodiff/](https://mroczect.github.io/velodiff/)

### 3. libcftext

A lightweight, pure C library (no dependencies) to determine whether a file contains text or binary data.

- Repository: [https://github.com/mroczect/libcftext](https://github.com/mroczect/libcftext)
- Docs: [https://mroczect.github.io/libcftext/](https://mroczect.github.io/libcftext/)

### 4. libcagekeygen

A minimal C library (no dependencies) to generate and convert X25519 key pairs for the age encryption tool.

- Repository: [https://github.com/mroczect/libcagekeygen](https://github.com/mroczect/libcagekeygen)
- Docs: [https://mroczect.github.io/libcagekeygen](https://mroczect.github.io/libcagekeygen)
