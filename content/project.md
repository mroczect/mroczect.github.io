# Projects

A collection of my open-source work focusing on systems programming, libraries, and developer tooling.

## Library & Systems Projects

### [velodiff](https://github.com/mroczect/velodiff)

A high-performance Rust library for computing diffs across Git repositories. It can diff commits, trees, and blobs, providing a programmatic interface ideal for building custom Git analysis and automation tools.

> **Topics:** `git`, `rust`, `diff`, `unix`, `library`  
> **License:** GNU General Public License v3.0

---

### [libcagekeygen](https://github.com/mroczect/libcagekeygen)

A minimal, dependency-free C library that generates X25519 key pairs for the **age** file encryption tool. It supports conversion between raw keys, age-formatted identities/recipients, and SSH ed25519 formats, making it easy to integrate age encryption into other C projects.

> **Topics:** `cryptography`, `clib`, `implementation`, `age`, `library`  
> **License:** GNU General Public License v3.0

---

### [rawssg](https://github.com/mroczect/rawssg)

A terminal‑themed static site generator written in Rust. It compiles Markdown files into a complete static site with a retro man‑page aesthetic. Features include a built‑in development server, live reload, syntax highlighting, and customizable themes.

> **Topics:** `rust`, `ssg`, `markdown`, `cli`, `static-site`  
> **License:** MIT License

---

### [libcftext](https://github.com/mroczect/libcftext)

A tiny, pure C library that accurately determines whether a file contains text or binary data using statistical analysis. Zero dependencies, works on any platform. Ideal for filters, viewers, and tools that need to safely process unknown files.

> **Topics:** `c`, `file-detection`, `text-binary`, `portable`  
> **License:** GNU General Public License v3.0

---

### [libcdaydiff](https://github.com/mroczect/libcdaydiff)

A comprehensive, portable C library for date and time manipulation. It handles parsing, formatting, arithmetic, and timezone-aware operations entirely in ANSI C with no external dependencies. Designed for embedded systems and cross-platform applications that need reliable date handling.

> **Topics:** `datetime`, `implementation`, `purec`, `clibrary`  
> **License:** GNU General Public License v3.0

---

### [snapcat](https://github.com/mroczect/snapcat)

A command-line tool that captures a snapshot of your project: directory tree, file contents (with syntax highlighting), and a unique fingerprint. Perfect for sharing project context in code reviews, documentation, or issue reports.

> **Topics:** `tree`, `cat`, `lib`, `cli`, `tools`  
> **License:** GNU General Public License v3.0

---

### [librawssg](https://github.com/mroczect/librawssg)

The core library behind [rawssg](https://github.com/mroczect/rawssg), providing the Markdown parsing, template rendering, and site generation engine. Under active development alongside rawssg.

> **Topics:** `rust`, `ssg`, `library`, `markdown`, `cli`  
> **License:** GNU General Public License v3.0
